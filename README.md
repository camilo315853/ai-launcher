## 📋 What is ai-launcher?

ai-launcher is a simple tool that helps you use AI programs directly from your computer's command line. It manages local files to keep your context, so you can work smoothly without needing an internet connection all the time. This tool is designed to support various AI models and coding helpers. It is made to run in your terminal on Windows and does not require deep technical knowledge to get started.

## 💻 System Requirements

To use ai-launcher on your Windows PC, your system must meet these conditions:

- Windows 10 or later (64-bit recommended)
- At least 4 GB of RAM
- Minimum 200 MB of free disk space
- Python 3.8 or higher installed (this is needed to run the program)
- A stable internet connection for initial setup and updates

If you do not have Python installed, you can get it from https://www.python.org/downloads/.

## 📝 How to Use ai-launcher

Using ai-launcher means typing simple commands in the terminal. Here are some straightforward examples:

- To start the assistant, type:  
  `ai-launcher start`  
- To ask a question, type:  
  `ai-launcher ask "Your question here"`  
- To see the help menu with options, type:  
  `ai-launcher help`  

Your commands will work with supported AI models that run on your local machine or connect to online services, depending on your setup.

## 📂 File and Context Management

ai-launcher saves data locally. It keeps track of your conversations, code snippets, and configurations to help provide better suggestions.

The default data location is:

`C:\Users\<YourUsername>\Documents\ai-launcher-data`

You can back up or move this folder if needed. This local approach keeps your data private and lets you work offline when needed.
